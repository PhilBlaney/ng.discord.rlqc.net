Role
Syntaxe:
.role [ID du rôle]
La commande role liste les membres ayant le rôle mentionné dans la commande.
Exemple:

.role 221298412782354433



Role Count
Syntaxe:
.rolecount [mot]
.rolecount ["plusieurs mots"]
La commande rolecount liste le(s) rôle(s) correspondant au nom recherché et affiche le nom des rôles, ainsi que le nombre de membres ayant ce(s) rôle(s).
Exemple:

.rolecount Rang
.rolecount "Rang C"



Role Purge
Syntaxe:
.rolepurge [id du rôle]
La commande rolepurge retire le rôle mentionné à tous les membres l'ayant. Toujours s'assurer que vous avez le bon ID dans Discord avant de rouler la commande. Vous pouvez taper "<@&ID du rôle>" dans discord avant votre commande pour voir quel rôle vous avez copié.Si le bon rôle est mentionné, vous pouvez ensuite entrer la commande rolepurge.
Exemple:

<@&682052895554011140> = @Rang C (muet)
.rolepurge 682052895554011140

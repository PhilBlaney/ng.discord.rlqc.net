Created
Syntaxe:
.created [ID]
La commande created mentionne la date et heure de la création de l'ID utilisé. L'ID peut être un ID de message, de canal textuel ou vocal, de rôle, d'utilisateur ou de serveur discord. La commande created marche pour tous les IDs discord, un ID non relié au serveur de Rocket League Québec va aussi retourner l'information.
Exemple:

.created 266009987354984449

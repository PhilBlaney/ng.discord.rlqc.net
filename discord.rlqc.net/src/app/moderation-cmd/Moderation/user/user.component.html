Leave
Syntaxe:
.u [ID d'un utilisateur]
.user [ID d'un utilisateur]
La commande user sort les informations de bases d'un utilisateur Discord. Par défaut, le nom, l'ID Discord, la date de création du compte ainsi que la photo de profil de l'utilisateur vont être affichés. S'il s'agit d'un ancien membre Discord, le nombre de mutes, la plus longue sanction, l'heure de sa dernière action sur le serveur, ses alias, ainsi que les liens utilisés pour rejoindre le Discord seront des informations affichées en plus.
Exemple:

.u 610588271805857816
.user 610588271805857816

Edit
Syntaxe:
.edit [ID du canal textuel] [ID du message] [message]
.edit [#canal-textuel] [ID du message] [message]
La commande edit permet de modifier un message envoyé via le Robot RLQC.
Exemple:

.edit 222427018040573953 900027899673989161 Hello there.
.edit #staff-commandes 900027899673989161 Ça part!



Edit With Embed
Syntaxe:
.embedimg [ID du canal textuel] [ID du message] [url d'une image]
.embedimg [#canal-textuel] [ID du message] [url d'une image]
La commande embedimg permet de modifier l'embed d'un message envoyé via le Robot RLQC.
Exemple:

.embedimg 222427018040573953 900027899673989161 https://rlqc.net/wp-content/uploads/2021/04/icon_live.png
.embedimg #staff-commandes 900027899673989161 https://rlqc.net/wp-content/uploads/2021/04/icon_live.png

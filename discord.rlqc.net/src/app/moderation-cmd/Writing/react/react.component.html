React
Syntaxe:
.react [id du canal] [id du message] [nom de l'emoji]
La commande react ajoute une réaction standard ou du serveur Rocket League Québec à un message, en tant que Robot RLQC.
Exemple:

.react 222427018040573953 900033660567318538 AngryCry
.react #staff-commandes 900033660567318538 AngryCry
